# file_rename
文件批量 改名系统\n
前端跑在8887 端口  后端跑在8888端口\n
指定根目录  然后可以搜索出 符合搜索条件的文件(包含文件夹)  \n
可以使用简单模式或者正则模式进行搜索  不写搜索条件默认 会搜索出所有文件\n
最后可以对搜索出的文件进行批量重命名操作 包括:增加前缀和去除相关字段\n
